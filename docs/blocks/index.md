# Blocks

Blocks 是 Yok UI 的组合示例区。它不是单个组件 API，而是把多个组件按真实业务任务组合成可复用页面片段，类似主流组件库里的模板、区块和 Pro 场景。

## How to read blocks

<div class="docs-card-grid">
  <section class="docs-card">
    <h3>Composition first</h3>
    <p>先理解信息结构和交互路径，再回到单组件页面查看具体 props。</p>
  </section>
  <section class="docs-card">
    <h3>Copy as pattern</h3>
    <p>区块示例用于沉淀页面模式，不直接替代业务代码中的数据请求和权限逻辑。</p>
  </section>
  <section class="docs-card">
    <h3>Promote reusable parts</h3>
    <p>当区块里的片段被多个场景复用时，再提升为 Product 或 Admin 组件。</p>
  </section>
</div>
